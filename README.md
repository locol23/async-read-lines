# async read lines

## Prerequisite

- [Yarn](https://yarnpkg.com/)
- [Node.js v16.13.0](https://nodejs.org/en/)

## Getting Started

```
yarn
yarn start
cat output.csv
```

## Notes

If you need a JavaScript file. You can get it by using the below command.

```
yarn build
ls -l dist
```

### Run JavaScript file

```
node dist/index.js
```
